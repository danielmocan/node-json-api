# node-json-api
Rest JSON api using node


## Usage

**Clone the repo**
```
git clone https://github.com/danielmocan/node-json-api.git
```
**Run project**
```
npm start
```
server will listen for requests on port 3000


### Routes

```
GET    /posts
GET    /posts/1
POST   /posts
PUT    /posts/1
DELETE /posts/1
```
### Adding Data
add it in the db.json file.
It has to be a valid json type.
